# Contributing

Thank you for considering contributing to the Media Manager!

## Code of Conduct

Please review and adhere to our Code of Conduct.

## Security Vulnerabilities

If you discover a security vulnerability within Media Manager, please send an e-mail to security@example.com. All security vulnerabilities will be promptly addressed.

## Pull Requests

1. Fork the repository and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Coding Standards

The Media Manager follows the PSR-1, PSR-2, and PSR-4 coding standards.
